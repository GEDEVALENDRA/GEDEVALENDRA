## ⚙️ Cara Kerja & Filosofi Coding

- **Berorientasi pada Solusi Nyata:** Bagi saya, kode terbaik adalah kode yang membawa manfaat. Saya senang memikirkan pengalaman pengguna—seperti menyisipkan *pop-up alert*, kotak dialog informasi, maupun logging terminal yang intuitif bagi pengguna awam.
- **Rapi & Terstruktur:** Sangat disiplin dalam manajemen repositori. Saya selalu memastikan folder proyek teratur, bebas dari file sampah biner, serta menggunakan konfigurasi `.gitignore` yang ketat demi menjaga kerahasiaan data sensitif (seperti API Keys / file `.env`).
- **Skalabilitas Tinggi:** Mengombinasikan kecepatan statis **Astro.js** dan fungsionalitas dinamis **React** di sisi frontend, serta ketangguhan arsitektur **Node.js/Express** dan **PHP** di sisi backend.
